# esp-wifi-rates
Type-safe Wi-Fi PHY rate handling for the ESP32-series chips.
## Usage
This is used by [esp-wifi-hal](https://crates.io/crates/esp-wifi-hal) for rate handling, and has been moved to its own crate, as it may be of general use.
